========================================
IF PUSH FAILS - TRY THESE
========================================

1. Check if remote is set correctly:
   git remote -v

2. If no remote, add it:
   git remote add origin https://github.com/AHADKHATTAK1/websiteecom.git

3. If wrong remote, update it:
   git remote set-url origin https://github.com/AHADKHATTAK1/websiteecom.git

4. If branch is different, check your branch:
   git branch
   
   Then push to your branch:
   git push origin <your-branch-name>

5. If rejected, pull first:
   git pull origin main --rebase
   git push origin main
