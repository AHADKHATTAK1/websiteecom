==========================================
VERCEL STORE DEPLOYMENT - QUICK STEPS
==========================================

PROBLEM: Store not showing on Vercel
REASON: Wrong root directory or not deployed

SOLUTION: Deploy with correct settings

==========================================
STEP 1: GO TO VERCEL
==========================================
https://vercel.com/dashboard

==========================================
STEP 2: NEW PROJECT
==========================================
Click: "Add New Project"
Select: Your GitHub repo (websiteecom)
Click: "Import"

==========================================
STEP 3: CONFIGURE (MOST IMPORTANT!)
==========================================

Framework: Vite

Root Directory: frontend/store
                 ^^^^^^^^^^^^^^
                 (Click Edit to set this!)

Build Command: npm run build

Output Directory: dist

Install Command: npm install

==========================================
STEP 4: DEPLOY
==========================================
Click: "Deploy"
Wait: 2-3 minutes
Done: You'll get URL!

==========================================
IF ALREADY DEPLOYED BUT NOT WORKING:
==========================================

1. Vercel Dashboard → Your Project
2. Settings → Root Directory
3. Change to: frontend/store
4. Save
5. Deployments → Redeploy

==========================================
TWO PROJECTS NEEDED:
==========================================

Project 1: Store
- Root: frontend/store
- URL: https://your-store.vercel.app

Project 2: Admin  
- Root: frontend/admin
- URL: https://your-admin.vercel.app

==========================================
CHECK: https://vercel.com/dashboard
==========================================

Detailed Guide: vercel_deployment_guide.md
